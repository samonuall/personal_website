# Getting Started #
npm run dev for dev server
npm test for running jest tests in testing folder

## Environment Variables ##

Copy `.env.example` to `.env.local` and configure:

- `NEXT_PUBLIC_PRIVATE_MODE`: Set to `true` to hide detailed information (e.g., when employed). Set to `false` or remove when looking for opportunities.
  - When `true`: Experience cards are not clickable and don't show details
  - When `false`: Full experience details are visible
